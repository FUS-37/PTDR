# PTDR
这是一个失真插件，0延迟，且cpu开销极小。这个插件用于拟合protools的未知summing带来的独特失真。

曾用名：protools风味失真

# 下载与安装
施工中

# 插件使用方法
目前这个插件是4个模式，bypass / soft / flat / harsh

其中flat是失真最像protools的，soft失真少一点，harsh失真多一点

使用方式1：
在混音开始前，在每个轨道都挂上，使用soft或者flat模式，用于模拟protools的失真的情况，给你带来熟悉的protools风味

使用方式2：
只给特定的轨道挂上这个效果器，可以使这个轨道对于其他轨道更加（干瘪/有力），是一个独特的给混音增加对比度的手法

使用方式3：
随便你怎么玩！
